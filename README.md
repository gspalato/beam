A YTDL wrapper with queue system for discord.js. 
<br/>

## 📦 Getting Started
### Prerequisites

-  [x] [npm](https://www.npmjs.com/)
-  [x] [node.js](https://nodejs.org/)
-  [x] [ffmpeg](https://ffmpeg.org/)

### Installing

Clone the repository to a folder and install dependencies.
```bash
$ npm i --save
```
